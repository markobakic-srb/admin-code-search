# Admin Code Search

Admin Code Search is a lightweight developer utility plugin for WordPress.

It allows administrators to search for text inside active theme and plugin files directly from the WordPress admin panel.

## Features

- Admin-only access
- Search active plugins
- Search active theme and parent theme
- Custom file extensions
- Line-by-line scanning
- Highlighted matches
- No external requests or tracking

## Installation

1. Download or clone this repository
2. Copy the `admin-code-search` folder into: wp-content/plugins/

3. Activate the plugin in WordPress
4. Go to **Tools → Code Search**
