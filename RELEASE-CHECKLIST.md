# Release Checklist

1. Update version in `admin-code-search.php`
2. Update `ADCOSE_VERSION`
3. Update `Stable tag` in `readme.txt`
4. Update changelog in `readme.txt`
5. Test plugin locally
6. Run Plugin Check if needed
7. Commit and push to GitHub
8. Create installable ZIP from `admin-code-search/`
9. Copy plugin files to SVN `/trunk/`
10. Commit trunk
11. Copy trunk to `/tags/x.x.x/`
12. Add and commit tag
13. Verify plugin page on WordPress.org
